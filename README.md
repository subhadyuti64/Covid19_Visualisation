# COVID-19 India Dashboard (Tableau)

## Overview

This project is an interactive Tableau dashboard that analyzes COVID-19 trends in India using multiple public datasets. The dashboard provides insights into confirmed cases, recoveries, deaths, vaccination progress, testing statistics, demographic distributions, and healthcare infrastructure.

The objective of this project is to transform raw COVID-19 data into meaningful visualizations that help users understand the impact, spread, and management of the pandemic across different states and demographics in India.

---

## Objectives

- Monitor the spread of COVID-19 across India.
- Analyze state-wise confirmed cases, recoveries, and deaths.
- Track vaccination progress and coverage.
- Understand demographic trends based on age and gender.
- Visualize testing statistics and healthcare infrastructure.
- Support data-driven insights through interactive dashboards.

---

## Dashboard Features

### 1. COVID Cases by State
- State-wise visualization of confirmed COVID-19 cases.
- Comparison of affected regions across India.
- Identification of hotspots and high-risk states.

### 2. Total Deaths Analysis
- Visualization of COVID-19 fatalities.
- State-level death statistics.
- Mortality trend analysis.

### 3. Age Group Analysis
- Distribution of cases among different age groups.
- Identification of vulnerable age categories.
- Demographic insights for healthcare planning.

### 4. Gender Analysis
- Gender-wise distribution of COVID-19 cases.
- Comparative analysis of infection rates among males and females.

### 5. Vaccination Dashboard

#### Doses Administered
- Total vaccine doses administered.
- State-wise vaccination progress.
- Coverage tracking across regions.

#### First Dose vs Second Dose
- Comparison of first and second dose administration.
- Analysis of vaccination completion rates.

### 6. State-wise Testing Analysis
- Number of tests conducted across states.
- Testing trends and coverage comparison.
- Evaluation of testing effectiveness.

### 7. ICMR Testing Labs
- Distribution of ICMR-approved testing laboratories.
- Analysis of healthcare testing infrastructure.
- Accessibility of testing facilities across India.
